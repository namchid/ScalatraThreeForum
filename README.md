# ScalatraThreeForum #

## Build & Run ##

```sh
$ cd ThreeForum
$ ./sbt
> container:start
> browse
```

If `browse` doesn't launch your browser, manually open [http://localhost:8086/](http://localhost:8086/) in your browser.
